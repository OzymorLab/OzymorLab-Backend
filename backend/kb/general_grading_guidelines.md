# General Subject Grading Guidelines

## 1. Core Grading Philosophy
- **Understanding Over Exact Phrasing**: Award marks if the student demonstrates clear understanding of the core concept. Do not penalize for missing exact textbook keywords unless the rubric explicitly mandates them.
- **Partial Credit and Step Marking**: Always award partial marks for correct intermediate steps, correct formulas, or sound logical arguments, even if the final conclusion or numerical answer is incorrect.
- **Error Propagation**: If a student makes a calculation error in Step A, but correctly uses that wrong value in Step B using the right methodology, award full marks for Step B. Do not penalize the student twice for the same mistake.

## 2. Subject-Agnostic Rules
- **Mathematical Derivations & Numericals**: 
  - The governing formula must be stated before substitution.
  - Minor algebraic skips are acceptable.
  - The final numerical answer must contain the correct SI unit. Deduct 0.5 marks for a missing or completely incorrect final unit.
- **Diagrams and Biology**:
  - *Ignore diagram evaluation*. Diagrammatic and graphical analysis is handled by a separate pipeline. If a rubric step explicitly asks to grade a drawn diagram, and no text accompanies it, award 0 marks with the justification: "Diagram evaluation is handled by an external module."
  - Biology specifics (like morphological drawings) are currently out of scope for this text-based evaluation.

## 3. Handling OCR Artifacts & Fuzzy Text
- Handwriting extraction can be fuzzy. If OCR misreads a handwritten symbol (e.g., confusing $I$ for $1$, $V$ for $v$, or $E = mc2$ for $E = mc^2$), use surrounding context to infer the student's intent. 
- Do not penalize for clear OCR limitations or messy handwriting as long as the underlying step is conceptually accurate.
